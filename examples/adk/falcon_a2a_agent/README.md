# 🛡️ Production Multi-Tenant isolated Secret Gateway

Falcon A2A agent using the Falcon MCP server in SaaS mode.

## ⚙️ Features

- **Tenancy Swaps Interceptor:** Decodes developer verification OKTA credentials or programmatically Live Profile Swaps proprietary Google workspace ya29. Opaque Access credentials resolving backend occupancy.
- **Isolated Tenancy isolated Secret Vault isolation routing:** Maps human occupancy identity keys providing automatic isolation routing stamping Vault Secret SEC_RES_NAME isolated connections routed backing isolated Tenant MCP runners.

## 🚀 Running Locally

Ensure that the MCP server is running in SaaS mode:

```bash
# Provide tenant isolation vaults mappings mapping.json
uv run uvicorn agent:a2a_app --port 8001
```
